# Event Registration System

A full-stack LAMP application demonstrating PHP, MySQL, and Apache.

## Features
- Event registration form with validation
- MySQL database with relationships
- Prepared statements for SQL injection prevention
- Responsive HTML/CSS design
- Display of all registrations

## Technologies
- Linux (Apache/XAMPP)
- Apache Web Server
- MySQL Database
- PHP Backend
- HTML/CSS Frontend

## How to Run
1. Install XAMPP
2. Clone this repository
3. Run the SQL setup script in phpMyAdmin
4. Place files in C:\xampp\htdocs\event-registration\
5. Navigate to http://localhost/event-registration/
